## Qué es Gemini

Familia de modelos de IA multimodal de Google DeepMind. Frontierline en comprensión multimodal, reasoning, y capacidades agénticas. Disponible como API (ai.google.dev) y como producto de consumo (gemini.google.com).

## Familia de modelos (estado 2026)

### Gemini 3 Pro (más capaz)
- LMArena Leaderboard: 1501 Elo (top 1 mundial al momento de publicación)
- Supera modelos anteriores en reasoning, multimodalidad y coding
- Soporta Computer Use tool (gemini-3-pro-preview)
- Orientado a tareas complejas, análisis avanzado, coding

### Gemini 3 Flash (equilibrio velocidad/calidad)
- Combina reasoning Pro-grade con latencia y costo Flash
- 3x más rápido que Gemini 2.5 Pro
- Precio: $0.50/1M input tokens, $3/1M output tokens
- Soporta Computer Use tool (gemini-3-flash-preview)
- Ideal para: análisis de video, extracción de datos, Visual Q&A

### Gemini 2.x (generación anterior, aún disponible)
- Gemini 2.0 Flash, Flash-Lite, Pro Experimental
- Introducida como "AI para la era agéntica" (dic 2024)

## Capacidades multimodales

Inputs soportados: texto, audio, imágenes, código, video.

**gemini-embedding-2-preview**: modelo de embeddings multimodal que soporta texto, imagen, video, audio y PDF en un solo modelo.

## Capacidades agénticas

- **Computer Use tool**: control de computadora (disponible en gemini-3-pro/flash-preview)
- Tool use: function calling nativo
- Long context windows: documentos largos sin chunking extremo
- Agentic workflows: citado como uno de los usos principales de Gemini 3 Flash

## Distinción: Gemini (app) vs Gemini API

- **gemini.google.com**: producto de consumo, chatbot accesible al público
- **ai.google.dev**: API para desarrolladores, acceso programático a los modelos
- Los modelos son los mismos; el acceso y pricing difieren
