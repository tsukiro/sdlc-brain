## Qué es CrewAI

Framework open-source de orquestación multi-agente. Permite construir y orquestar equipos colaborativos de agentes de IA. Énfasis en accesibilidad y velocidad de prototipado.

## Conceptos núcleo

### Agents (Agentes)
Entidades de IA configurables por: rol, objetivo, historial. El desarrollador elige LLM, habilita razonamiento/memoria, asigna herramientas. Definibles en YAML, código o ambos.

### Tasks (Tareas)
Unidades de trabajo con: descripción, salida esperada (markdown/JSON/objeto), guardrails opcionales, HITL (Human-in-the-Loop) opcional.

### Crews (Equipos)
Conjunto de agentes + tareas que colaboran. Puede incluir: manager agent que orquesta el flujo, método de colaboración configurable, planificación opcional, delegación entre agentes.

## Características

**Planificación**: Agente gestor crea un plan paso a paso para todas las tareas antes de ejecutar.

**Razonamiento**: Agentes reflexionan sobre objetivos, crean y refinan planes estructurados.

**Herramientas**: Cientos de herramientas open-source (búsqueda web, interacción con sitios, consultas a bases de datos vectoriales).

**Memoria**:
- Corto plazo compartida (entre agentes del mismo crew)
- Largo plazo (persistente entre runs)
- Entidades (knowledge graph de entidades mencionadas)
- Contextual (resúmenes de conversaciones pasadas)

**Conocimiento (Agentic RAG)**: Combina fuentes diversas (archivos, sitios web, vector DBs) con reescritura inteligente de consultas.

**Trazabilidad**: Trazas disponibles para crews y flows locales.

## Flujo de desarrollo

1. Definir Agentes (roles, objetivos, herramientas, LLMs)
2. Definir Tareas (trabajo + resultados esperados)
3. Definir el Crew (combinar agentes + tareas, elegir colaboración)
4. Ejecutar

## Tiers de producto

| Tier | Descripción |
|------|-------------|
| CrewAI OSS | Framework open-source gratuito |
| CrewAI AMP Cloud | SaaS: editor visual, despliegue, escalado |
| CrewAI AMP Factory | Enterprise: on-prem o VPC privada |

## Casos de uso documentados

- Automatización de llamadas y recordatorios
- Análisis de acciones con múltiples agentes
- Selección y comparación de opciones (ciudades, productos)
- Recepcionistas virtuales / consultoría legal

## Comparativa con LangGraph (según benchmarks comunidad 2026)

- Prototipado: ~10 min (CrewAI) vs ~45 min (LangGraph)
- CrewAI → prototipado rápido, abstracción accesible
- LangGraph → producción, state management avanzado, control preciso
