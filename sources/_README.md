# sources/

Esta carpeta contiene los documentos fuente originales del wiki.

**Reglas:**
- El LLM nunca modifica ni elimina archivos aquí
- Añade aquí tus fuentes crudas: artículos, PDFs exportados como texto, notas, transcripciones, datos
- Después de añadir un archivo, dile al LLM "ingest [nombre del archivo]"

**Formatos admitidos:** `.md`, `.txt`, `.csv`, cualquier texto plano

**Convención de nombrado:** `YYYY-MM-DD_titulo-descriptivo.md` (la fecha ayuda a mantener el log coherente)
